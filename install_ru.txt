Интеграция скрипта Sypex Dumper 2.0.x в админ панель MODx 1.0.x
========================

Установка
=========

Вы можете выбрать два варианта установки.
1. Патч (более простой в установке, но при обновлении MODx, возможно нужно будет повторно залить патченный файл)
2. Модуль (сложнее в установке, но не требует переустановки при обновлении MODx)


Установка как Патч (замена модуля резервного копирования MODx)
=========
1. Распаковать содержимое архива SypexDumper_208.zip в каталог modx
2. Распаковать папки "manager" и "sxd" из архива в каталог modx (с заменой файлов)
3. Установить права на запись серверу для папки backup в каталоге sxd (modx/sxd/backup) => 0777
4. Установить chmod 666 для файлов modx/sxd/cfg.php и modx/sxd/ses.php.
5. Зайти в админку MODx, меню Tools -> Backup в котором можно делать бэкап/восстановление базы в Sypex Dumper 2 без дополнительной авторизации.


Установка как Модуль (Установка модуля MODx)
=========
1. Распаковать содержимое архива SypexDumper_208.zip в каталог modx
2. Распаковать папки "sxd" из архива в каталог modx (с заменой файлов)
3. Установить права на запись серверу для папки backup в каталоге sxd (modx/sxd/backup) => 0777
4. Установить chmod 666 для файлов modx/sxd/cfg.php и modx/sxd/ses.php.
5. Зайти в админку MODx -> Modules и создать модуль, заполнить поля

Название: Sypex Dumper 2
Описание: Backup and restore MySQL
Включить блокировку только для админов

В код модуля добавить содержимое файла module_code.php

6. Зайти в админку MODx, меню Modules -> Sypex Dumper 2 в котором можно делать бэкап/восстановление базы в Sypex Dumper 2 без дополнительной авторизации.
